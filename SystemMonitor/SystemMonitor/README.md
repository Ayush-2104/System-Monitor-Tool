# System Monitor Tool

## Overview
A console-based system monitor tool built in C++ for Linux. It displays real-time CPU load, memory usage, and active processes, similar to the `top` command.

## Features
- Real-time CPU and memory stats
- Process list with PID, name, CPU and memory usage
- Refreshes every 2 seconds
- Modular design using C++ classes
